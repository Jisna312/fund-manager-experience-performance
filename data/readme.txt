This folder should contain the following data files:

1. crspm.sas7bdat  → CRSP Survivor-Bias-Free US Mutual Fund database (data can't be shared due to proprietary reasons)
2. risk_free.csv   → The Monthly Frequency Fama French factors and Momentum factor from Jan 1992
3. CRSPM_JOIN.csv  → Cleaned dataset created by data_cleaning.R

These files are not included in this repository. 
Please place them here locally before running the scripts.
